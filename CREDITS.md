## Sound Effects

The sound effects in `assets/sounds/` were **synthesized from scratch** for
this project. They are dedicated to the public domain under
[CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).

| File | Purpose | Description |
|------|---------|-------------|
| `move.wav` | Piece moved | Short 220 Hz + 440 Hz click, ~140 ms |
| `capture.wav` | Piece captured | Low 80 Hz thunk + noise burst, ~180 ms |
| `check.wav` | King in check | Bright 880 Hz / 1320 Hz tone with vibrato, ~220 ms |
| `castle.wav` | Castling | Soft C5 + G5 sine chord, ~320 ms |
| `game_start.wav` | Game begins | Short E5 + A5 greeting ping, ~180 ms |
| `game_end.wav` | Game ends | Rising C5-E5-G5-C6 arpeggio, ~580 ms |

All sounds are 16-bit PCM mono WAV at 44.1 kHz.

### Reproducing the Sounds

The sounds are generated by the script
[`assets/sounds/_generate_sounds.py`](assets/sounds/_generate_sounds.py),
which only depends on `numpy` and the Python standard library. To regenerate
the WAV files, run:

```bash
python assets/sounds/_generate_sounds.py
```

> Earlier versions of this project bundled sound files from chess.com. Those
> files have been removed; the CC0 originals above replace them.

## Algorithms & Code References

### Accuracy Percent Formula
The game-accuracy calculation in `src/backend/analyzer.py` is based on the
formula used by Lichess:

- Source: [lichess-org/lila -- AccuracyPercent.scala](https://github.com/lichess-org/lila/blob/master/modules/analyse/src/main/AccuracyPercent.scala)

### Chess.com PGN Format
Time-control parsing for chess.com-style `[%clk H:MM:SS.s]` clock comments
in `src/backend/pgn_parser.py` is informed by the public
[PGN specification](https://www.chessclub.com/help/PGN-spec) and the
[chess.com PGN export format](https://www.chess.com/terms/pgn).

### Stockfish Engine
Analysis is performed by the [Stockfish](https://stockfishchess.org/)
chess engine, licensed under GPLv3. Stockfish is **not bundled** with
this repository -- users supply their own binary, and the engine path is
configured via the application settings.
